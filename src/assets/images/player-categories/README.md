# Player Category Icons

This directory contains 32x32 PNG icons for cricket player categories:

- **allrounder.png** - Star icon (multi-skilled player)
- **batting-allrounder.png** - Bat with star overlay
- **batsman.png** - Cricket bat icon
- **bowling-allrounder.png** - Ball with star overlay  
- **bowler.png** - Cricket ball icon
- **keeper-batter.png** - Wicket keeper gloves with bat
- **commentary.png** - Microphone icon
- **fielding.png** - Hand/glove catching
- **nothing.png** - Dash or empty icon
- **unknown.png** - Question mark icon

## Icon Requirements
- Size: 32x32 pixels
- Format: PNG with transparency
- Style: Simple, clear, recognizable cricket symbols
- Background: Transparent

## Usage
Icons are displayed as overlays on player images in the bottom-right corner.
